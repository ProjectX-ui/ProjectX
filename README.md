# 🚀 ProjectX

Официальный репозиторий проекта **ProjectX**. 
Здесь вы можете получить доступ к инструментам мониторинга и управления.

## 🤖 Запустить бота
@rcpcfilebot
просто введи в поиск телеграмм юзернейм бота
---

### 🛠 Основные возможности:
* **Авто-выдача:** Получение исходного кода в один клик.
* **Безопасность:** Работа через защищенные протоколы.
* **Поддержка:** Прямая связь с разработчиком через интерфейс бота.
* **Реферальная система:** Бонусы за приглашение друзей.

### 📋 Как использовать:
1. Перейдите в бота по ссылке выше.
2. Нажмите `/start`.
3. Используйте меню для выбора нужной версии софта(Бесплатной и платной).
